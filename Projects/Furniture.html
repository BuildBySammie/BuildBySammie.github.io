Furniture Design

Building has always been a big part of my live and welding has become one of my most favourite way to build. My father taught me to weld using electrodes when I was twelve and I have been welding ever since. My first project was a wood burner but now I mostly create indoor and outdoor furniture for family and friends.


My first commision

My first big commission was a dining room set which was my first welding project with gasless MIG. The set consists out of one big table with a large bench and a table extender with two stools. The extender can be used to enlarge the size of the table (as the name creatively suggests) so dinner parties can be held with more people. But if it is not needed it can be used as a side table and the stools can be stored between the legs.


Audio Rack

My latest project was an audio rack. The goal was to tidy up my audio equipment and give them a great home. I wanted a design that looked like it was a single piece of steel snaking around the shelfs. To accomplish this I used 20 by 20mm square tube steel that was cut in such a way that all the manufacturing marks of the steel tube would flow as a single line. Which created the appearance of a single tube flowing around the shelfs.
